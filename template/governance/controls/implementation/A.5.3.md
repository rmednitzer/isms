# Implementation statement for A.5.3 Segregation of duties

**REC-2026-503 Revision 1 (DRAFT)**

*Status: draft. Owner: role:ISMS-Manager. Next review: 2027-01-01.*

## Control

A.5.3 Segregation of duties (theme: organisational).

## Implementation

Conflicting duties are separated per `users/separation-of-duties.yaml`; where segregation is impractical for a small organisation, compensating controls and the documented exception apply.

## Evidence

No dedicated automated evidence task is bound to this control yet; evidence is captured manually per SOP-301 and the binding is tracked in `controls/evidence-plan.yaml`.
